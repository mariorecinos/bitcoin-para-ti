# Bitcoin Para TI

A simple web application that displays cryptocurrency prices in different fiat currencies using the Coinbase API

# Technologies Used:

* HTML 5
* CSS
* Bootstrap
* Google Fonts
* Javascript
* jQuery
* Coinbase API
*New Changes
